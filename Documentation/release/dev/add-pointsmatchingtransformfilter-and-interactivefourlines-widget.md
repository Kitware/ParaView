## Add PointsMatchingTransformFilter in filters list and an InteractiveFourLines widget to interact with

The `PointsMatchingTransformFilter` transform any vtkPointSet by specifying 4 pairs of points in space.

To interact with this filter, there is a new `InteractiveFourLines` widget to control the source and the target points of each line. It is also possible to switch the source point and the target point of a line.
